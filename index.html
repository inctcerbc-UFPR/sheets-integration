📋 Como Testar

1. Criar Planilha

Acesse Google Sheets e crie uma nova planilha

2. Adicionar Dados

Na primeira linha (cabeçalho):

Nome | Instituição

Depois adicione alguns dados:

Prof. João | UFPR
Prof. Maria | UFMS
Prof. Pedro | UNIFESP

3. Compartilhar

Clique em "Compartilhar" → Selecione "Qualquer pessoa com o link"

4. Copiar ID

A URL fica assim:

https://docs.google.com/spreadsheets/d/1ABC123DEF456GHI789/edit

Copie a parte em negrito (o ID)

5. Configurar

Abra o arquivo sheets-config-simple.js e altere:

GOOGLE_SHEET_ID: 'SEU_ID_AQUI'

Para:

GOOGLE_SHEET_ID: '1ABC123DEF456GHI789'

6. Upload

Faça upload destes arquivos para GitHub Pages:

  • index-simple.html → renomear para index.html
  • sheets-config-simple.js
  • sheets-parser-simple.js
  • styles.css
  • script.js

7. Testar

Abra seu site e verifique:

  • Abra o Console (F12)
  • Procure por "[Sheets]" nos logs
  • Edite algo na planilha
  • Aguarde 10 segundos
  • Veja os dados atualizarem! 🎉
